Changelog

История версий и изменений

v0.1 29 марта 2026 Текущая версия

Первый публичный релиз. Базовый функционал для создания и ведения археологических описей.

  • Конструктор таблицы — задавайте свои правила для каждого поля
  • 7 типов данных: текст, целое число, число с индексом, дробное, логическое, дата, список
  • Подсветка ошибочных записей с расшифровкой ошибки
  • Интуитивный фильтр с сохранением и загрузкой
  • Полевое кодирование — расшифровка сокращений (Q → кварц)
  • Импорт данных из CSV
  • Экспорт в CSV и отчёт DOCX
  • Быстрая сводка по отображаемым записям
  • Сессии работы
  • Русский интерфейс
  • Работа без установки, ~60 МБ, поддержка запуска с флешки

Следите за обновлениями — следующие версии уже в разработке
