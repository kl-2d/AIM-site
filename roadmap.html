Планы

Что разрабатывается для следующих версий

в планах

Археологический каталог

Позволяет сохранять и переиспользовать все значения, столбцы, коды и правила, которые вы вводите в таблицу. При создании или редактировании таблицы выбирайте нужные из каталога, чтобы не создавать их заново и экономить время.

в планах

Множественные таблицы

Несколько таблиц в одной сессии со сквозным фильтром между ними. Удобно для работы с разными типами таблиц в рамках одного раскопа.

в планах

Генератор этикеток

Автоматическое создание этикеток для артефактов из данных активной таблицы. Печать прямо из программы.

в планах

Расширенная работа с данными

Выпадающие списки, сниппеты, подсказки при вводе, групповой ввод для нескольких строк одновременно.

в планах

Горячие клавиши

Хоткеи для частых операций — ускорит работу при интенсивном вводе данных в полевых условиях.

в планах

Экспорт структуры таблицы

Выгружаемое описание структуры и правил таблицы — для документирования методики и передачи между экспедициями.

в планах

Расширение типов данных

Новые специализированные типы для археологических данных и конструктор собственного типа данных.

в планах

Совместная работа

Одновременная работа нескольких исследователей над одной таблицей через сервер. Актуально для крупных экспедиций.
