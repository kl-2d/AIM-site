FAQ

Частые вопросы и ответы

Чем AIM отличается от Excel, Access, ГИС, базы данных и т.д.?

AIM — это готовое решение для заполнения археологической таблицы. Программа имеет простой интерфейс и не требует специальных навыков.

Позволяет быстро организовать безошибочный, единообразный, контролируемый процесс ведения таблицы по заданным вами правилам.

Её разработка преследует цель организовать как можно более удобно процесс сбора данных для самого широкого круга пользователей. Возможно, это первая такая разработка на территории России, которая доведена до готовности.

Для кого эта программа?

AIM предназначена для археологов и исследователей, которые ведут полевые описи артефактов. Подойдёт как для небольших раскопок, так и для крупных экспедиций с большим объёмом данных.

Нужно ли что-то устанавливать?

Нет. AIM работает без установки — просто распакуйте архив и запустите. Размер программы около 60 МБ. Можно запускать прямо с флешки на любом рабочем месте.

Нужен ли интернет для работы?

Нет. AIM работает полностью офлайн — это важно для полевых условий на раскопе.

Какие операционные системы поддерживаются?

На данный момент — Windows. Поддержка других платформ планируется в будущих версиях.

Можно ли работать с данными из Excel?

Да. Сохраните таблицу Excel как CSV и импортируйте в AIM. Экспортированные данные также открываются в Excel. Прямой импорт XLSX планируется в следующих версиях.

Сколько записей поддерживает программа?

AIM протестирован на таблицах с 1 000 000 записей без потери производительности. Для большинства полевых задач этого более чем достаточно.

Программа бесплатная?

Да, AIM полностью бесплатна и распространяется как freeware. Если хотите поддержать развитие — смотрите раздел на странице скачивания.

Как передать настройки фильтра коллеге?

Сохраните фильтр в файл и передайте его — коллега загрузит его в свою копию программы. Фильтры хранятся в виде обычных JSON-файлов.

Что такое полевое кодирование?

Возможность вводить сокращения вместо полных названий прямо в поле. Например, вместо «кварц» можно набрать Q — программа расшифрует это в полное значение. Ускоряет ввод данных на раскопе.
