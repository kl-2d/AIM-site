Что такое AIM?

AIM — настольная программа для археологов, которая помогает вести структурированные описи найденных артефактов прямо в поле или в лаборатории.

Вместо разрозненных таблиц Excel — единый инструмент с валидацией данных, гибкой фильтрацией и генерацией отчётов. Поддерживает работу с большими объёмами: до миллиона записей без потери скорости.

Возможности

📋

Гибкая структура таблиц

Создавайте таблицы с любым набором колонок. Семь типов данных: текст, число, дата, логическое значение, список и другие.

Валидация данных

Встроенная проверка на ошибки при вводе. Программа сразу показывает какие записи некорректны и почему.

Мощная фильтрация

Фильтруйте по любым полям — текст, диапазоны чисел, даты, списки значений. Сохраняйте фильтры для повторного использования.

📥

Импорт CSV

Загружайте данные из CSV-файлов. Гибкое сопоставление колонок файла с колонками таблицы.

Экспорт отчётов

Сессии работы

Сохраняйте и открывайте рабочие сессии. Конфигурация таблицы и данные хранятся в удобном формате.

FAQ

Для кого эта программа?

AIM предназначена для археологов и исследователей, которые ведут полевые описи артефактов. Программа подойдёт как для небольших раскопок, так и для крупных экспедиций с большим объёмом данных.

Какие операционные системы поддерживаются?

На данный момент — Windows. Поддержка Linux и macOS планируется в будущих версиях.

Нужен ли интернет для работы?

Нет. AIM работает полностью офлайн — это важно для полевых условий.

Можно ли импортировать данные из Excel?

Сохраните таблицу Excel как CSV и импортируйте в AIM. Прямой импорт XLSX планируется в следующих версиях.

Сколько записей поддерживает программа?

AIM протестирован на таблицах с 1 000 000 записей. Для большинства задач производительность будет достаточной.

Программа бесплатная?

Да, AIM полностью бесплатна.
